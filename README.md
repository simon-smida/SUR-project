## Dependencies
1. Create virtual environment
```bash
python3 -m venv venv
```
2. Activate the environment
```bash
source ./venv/bin/activate
```
3. Install dependencies
```bash
pip install -r requirements.txt
```

## Data Augmentation
From the main directory do:
```bash
python3 ./src/dataAugmentation.py
```
